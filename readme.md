### CURE.exe - Color-Unicoded moRE.exe

Text Viewer

- Running on Windows-Command-Prompt.
- Support UTF8 text
- Support ANSI Escape Sequences.

## Usage

    CURE.exe [-b] {filename(s)}
    CURE.exe -h
    PROGRAM | CURE.exe [-b]

# Option

* `-b` : Set bold on each line.
* `-h` : Help

## How to build

    make.cmd get
    make.cmd
